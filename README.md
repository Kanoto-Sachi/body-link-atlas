# Body Link Atlas — 公開リポジトリ向け強化ロック版 v2

GitHub Pagesで公開できる、HTML / CSS / JavaScriptのみの静的Webアプリです。

## v2の追加機能

- 物質リンクビュー
- 分野横断ビュー
- カタカナ語・略語・検査値・ホルモン・神経伝達物質などのジャンル整理
- 関連マップで物質から病態・検査・看護観察へ移動

## セキュリティ方針

- 公開リポジトリには `data/encrypted-data.json` のみを置きます。
- 平文の `keywords.json` / `relations.json` は公開しません。
- 閲覧時に入力したパスワードで、ブラウザ内で復号します。

## 注意

GitHub Pagesは静的サイト公開です。本格的な会員制ログインではありません。
個人情報、患者情報、学校名、非公開資料そのものは入れないでください。

医学的内容は学習用の下書きです。AI生成サンプルには「要確認」タグを付け、教科書・授業資料で確認してください。
