# Security Policy

## 公開してよいファイル

- `index.html`
- `app.js`
- `style.css`
- `README.md`
- `SECURITY.md`
- `.gitignore`
- `data/encrypted-data.json`
- `tools/encrypt.html`

## 公開しないファイル

- `keywords.json`
- `relations.json`
- `body-link-atlas-private-data-bundle.json`
- `private-data-expanded/`
- パスワードを書いたメモ
- 個人情報、患者情報、学校名、非公開資料

## 注意

このロックはGitHub Pages向けの静的サイト用です。本格的なサーバー認証ではありません。
データは暗号化されていますが、パスワードが弱い場合や公開された場合は保護できません。
